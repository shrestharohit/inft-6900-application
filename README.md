# How to run guide:
1. Clone the repository in your local device 
> `git clone https://github.com/shrestharohit/inft-6900-application.git`
2. Use `yarn` package managers to load the node_modules from package.json
> `yarn install`
3. setup the `.env` file in the main path in the directory and copy the contents from `.env.sample`, (all secret credentials are in report, you will need these for the backend port) 
4. Then simply run the project
>   `yarn dev`
